# 要件定義・基本設計】教育系eラーニングシステム新規開発／AWS・React・Next.js・Python

仕事の詳細
案件概要
大手教育研修サービス企業における、通信教育・eラーニング統合プラットフォームの新規構築プロジェクトです。
既存の受講管理・申込・基幹システム・添削運用などのレガシーシステムを統合し、新しいプラットフォームを段階的に構築します。
今回は要件定義・基本設計を担当いただく上流SEを募集します。

業務内容
顧客（事業部門・システム部門）へのヒアリング
業務要件の整理・要件定義書の作成
基本設計書（画面設計・DB設計）の作成
外部システムとの連携仕様・インターフェース設計
インフラベンダー・デザイン会社との仕様調整
現行システムの調査および移行要件の整理
定例会・課題管理
開発チームへの要件連携
